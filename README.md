# Student Performance Analysis 📊

## 📌 Overview
This project analyzes student performance data to identify key factors affecting academic results.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib / Seaborn

## 📂 Dataset
The dataset contains information about students such as:
- Study hours
- Attendance
- Scores

## 📊 Key Insights
- Students studying more than 6 hours scored higher marks
- Low attendance (<70%) leads to poor performance
- Math scores show higher variation compared to other subjects

## 📈 Visualizations
- Bar graphs
- Heatmap (correlation)
- Distribution plots

## 🚀 Conclusion
This analysis helps in understanding how different factors impact student performance and can be used to improve learning strategies.
